Aspect Ratio Component Examples

This directory contains copy-pasteable HTML examples for aspect ratio containers. Each example can be found in its own HTML file:

  • AspectRatio1-1.html - 1:1 (Square)
  • AspectRatio16-9.html - 16:9 (Landscape Video)
  • AspectRatio9-16.html - 9:16 (Portrait Video)
  • AspectRatio3-2.html - 3:2 (Landscape Photo)
  • AspectRatio2-3.html - 2:3 (Portrait Photo)

CSS Classes

Container: .aspect-ratio

Ratios:
  .aspect-ratio--1-1    (1:1 - Square)
  .aspect-ratio--16-9   (16:9 - Landscape Video)
  .aspect-ratio--9-16   (9:16 - Portrait Video)
  .aspect-ratio--3-2    (3:2 - Landscape Photo)
  .aspect-ratio--2-3    (2:3 - Portrait Photo)

Content: .aspect-ratio__content

Usage Example

<div class="aspect-ratio aspect-ratio--16-9">
  <div class="aspect-ratio__content">
    <!-- Your content here -->
  </div>
</div>
