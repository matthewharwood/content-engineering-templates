Accessibility Features

  • Semantic HTML: Uses native <select> element for full keyboard support
  • ARIA Labels: Proper aria-describedby, aria-required, and aria-invalid attributes

  • Focus States: Clear visual focus indicators meeting WCAG 2.1 AA standards
  • Color Contrast: All text and borders meet WCAG AA contrast requirements
