Accessibility Features

  • role="switch" - Properly identifies the control as a switch
  • aria-checked - Indicates current state to screen readers
  • aria-describedby - Links descriptions to switches
  • aria-required - Marks required fields
  • aria-invalid - Indicates validation errors
  • Keyboard support - Space toggles, Tab navigates
  • Focus indicators - Clear outline on keyboard focus
  • Semantic HTML - Fieldset/legend for grouped switches
