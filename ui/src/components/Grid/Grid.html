Grid System Examples

This directory contains copy-pasteable HTML examples for the 12-column grid system. Each example can be found in its own HTML file:

  • TwelveColumns.html - Basic 12 column grid
  • ColumnSpans.html - Column span utilities (col-span-3, col-span-6, etc.)
  • ResponsiveLayout.html - Simple sidebar + main layout
  • HolyGrailLayout.html - Desktop-only Holy Grail layout
  • HolyGrailResponsive.html - Fully responsive Holy Grail layout
