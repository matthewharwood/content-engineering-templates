Accessibility Features

  • Native HTML: Uses <details> and <summary> elements for built-in accessibility
  • Keyboard Navigation: Tab to navigate, Enter or Space to toggle
  • Screen Reader Support: Automatic state announcements (expanded/collapsed)
  • Focus Indicators: Clear focus outlines for keyboard users
  • Color Contrast: All variants meet WCAG 2.1 AA standards
  • Animation: Smooth transitions with prefers-reduced-motion support
  • Semantic HTML: Proper heading hierarchy and content structure
