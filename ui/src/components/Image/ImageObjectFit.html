Object-Fit Modes

The object-fit CSS property determines how an image should be resized to fit its container. Each mode handles aspect ratio preservation and cropping differently.

object-fit: cover

Fills container, may crop

object-fit: contain

Entire image visible, letterbox

object-fit: fill

Stretches to fill, may distort

object-fit: none

Original size, may overflow

object-fit: scale-down

Smaller of none or contain
