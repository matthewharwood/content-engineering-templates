Typography Examples

This directory contains copy-pasteable HTML examples for typography. Each example can be found in its own HTML file:

  • AllLevels.html - All type scale levels (Step -4 through Step 8)
  • Headings.html - Semantic heading tags (h1-h6)
  • BodyText.html - Body text examples at different scales
