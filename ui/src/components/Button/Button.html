Button Component Examples

This directory contains copy-pasteable HTML examples for buttons. Each example can be found in its own HTML file:

  • ButtonPrimary.html - Primary button kind
  • ButtonSecondary.html - Secondary button kind
  • ButtonTertiary.html - Tertiary button kind (no background until hover)
  • ButtonOutline.html - Outline button kind
  • ButtonSizes.html - All button sizes (mini, small, medium, large)
  • ButtonDisabled.html - Disabled state
  • ButtonPill.html - Pill shape (fully rounded ends)
  • ButtonCircle.html - Circle shape (for icons, numbers, letters)
  • ButtonShapes.html - All button shapes
  • ButtonWithIcons.html - Buttons with start and end icon enhancers
  • ButtonShowcase.html - Complete showcase of all variants

CSS Classes

Base: .btn

Button Kinds:
  .btn--primary
  .btn--secondary
  .btn--tertiary
  .btn--outline

Sizes:
  .btn--mini
  .btn--small
  .btn--medium
  .btn--large

Shapes:
  (default - 4px border-radius)
  .btn--pill
  .btn--circle

Icon Enhancers:
  .btn--with-start-icon
  .btn--with-end-icon

Icon Button Usage

<!-- Start Icon -->
<button class="btn btn--primary btn--medium btn--with-start-icon">
  <span class="icon">add</span>
  Create New
</button>

<!-- End Icon -->
<button class="btn btn--secondary btn--medium btn--with-end-icon">
  Continue
  <span class="icon">arrow_forward</span>
</button>

<!-- Icon Only (Circle) -->
<button class="btn btn--primary btn--circle btn--medium">
  <span class="icon">search</span>
</button>
